# 路上英语后端

这个小后端用来保存云端词库。前端用“后端地址 + 同步码”上传和恢复单词。

## 本地试用

```bash
cd backend
npm install
npm start
```

本地后端地址是：

```text
http://localhost:8787
```

## 正式长期保存

推荐部署到 Vercel，并绑定 Upstash Redis 数据库。这样手机不依赖电脑，清浏览器缓存后也能用同步码恢复。

部署后，把 Vercel 给你的地址填到 App 的“设置 -> 云端词库 -> 后端地址”。

同步码自己生成并保存好。换手机时，填同一个后端地址和同步码，点“恢复”。
